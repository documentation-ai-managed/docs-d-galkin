---
title: Cannot Upload File
description: How to fix upload problems in Di.G.
---

## Symptoms

- Upload button does not respond
- Error: "File upload failed"

## Causes

- File size exceeds 20MB limit
- Unsupported format

## Solution

<Steps>
  <Step title="Check file size">
    Ensure your file is under 20MB.
  </Step>
  <Step title="Supported formats">
    Only .jpg, .png, and .pdf files are supported.
  </Step>
  <Step title="Refresh page">
    Try again after reloading Di.G interface.
  </Step>
</Steps>
