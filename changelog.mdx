## Latest Updates

<Update label="2024-10-15" description="v1.2.0" tags={["feature", "improvement"]}>

## New Features

- Added native integrations with Slack and Zapier for seamless workflow automation.
- Introduced webhook support for real-time event notifications to your custom endpoints.

## Improvements

- Enhanced dashboard performance with optimized data loading, reducing load times by `>40%`.
- Improved error handling in API responses with more descriptive messages.

## Bug Fixes

- Fixed intermittent failures in batch processing endpoints.
- Resolved UI glitches on mobile devices during theme switching.

</Update>

<Update label="2024-09-20" description="v1.1.0" tags={["improvement", "bugfix"]}>

## Improvements

- Redesigned the analytics dashboard with customizable widgets and dark mode support.
- Upgraded authentication flows to support multi-factor authentication (MFA).

## Bug Fixes

- Patched security vulnerability in token refresh mechanism (CVE-2024-5678).
- Corrected data export issues when handling large datasets (`>10,000` records).

</Update>

## Upgrade Instructions

<Steps>
  <Step title="Check Compatibility" icon="check-circle">

Review the [migration guide](/quickstart) for breaking changes in your version.

  </Step>
  <Step title="Update Package" icon="download">

```bash
npm install d-galkin@latest
```

Or for Python:

````python
pip install d-galkin==1.2.0
````
